## ⚡ 기술 요구사항

### 📐 **이미지 최적화**
- **포맷**: JPG (사진), WebP (지원 시)
- **품질**: 80-90% 권장
- **용량**: 개당 500KB 이하 권장

### 🎨 **촬영 품질**
- **해상도**: 최소 1000px 이상
- **조명**: 자연광 + 실내조명 조합
- **노출**: 밝고 선명하게
- **구도**: 수평/수직 정렬

## 🚀 적용 방법

1. **이미지 촬영 완료 후**:
   ```bash
   # 프로젝트 루트에 images 폴더 생성
   mkdir images
   
   # 촬영한 사진들을 해당 파일명으로 저장
   cp 촬영사진1.jpg ./images/fitness055-yongho-hero.jpg
   cp 촬영사진2.jpg ./images/freeweight-zone.jpg
   # ... 나머지도 동일
   ```

2. **자동 적용**: 
   - 파일명만 정확하면 자동으로 적용됩니다
   - fallback 이미지가 설정되어 있어 안전합니다
